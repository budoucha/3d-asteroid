# 3D ASTEROID

赤青3Dメガネ対応のベルトスクロールアクションゲームです。画面奥から迫ってくる隕石を避け続け、スコアを伸ばします。

## 遊び方

ブラウザで `index.html` を開くとプレイできます。

公開版:

https://budoucha.github.io/3d-asteroid/

## 操作

| 操作 | キー / 入力 | スマホ / タッチ |
|---|---|---|
| 移動 | 矢印キー / WASD / ドラッグ | ドラッグ / 左下スティック |
| スタート / リトライ | Space / クリック | 画面タップ |
| 一時停止 | Space | 下部の一時停止ボタン |
| 加速 | Z | BOOSTボタンを押し続ける |
| 低速移動 | Shift | SLOWボタンを押し続ける |
| 3D / 2D 表示切替 | G | 設定内の 3D / 2D ボタン |
| 立体感の調整 | `+` / `-` | 設定内の `+` / `-` ボタン |

赤青3Dメガネを使う場合は、左目が赤、右目がシアンのフィルターになる向きで装着してください。

## ファイル構成

```text
.
├── index.html              # ゲーム本体
├── assets/
│   └── twitter-card.png    # OGP / Twitterカード画像
└── docs/
    └── depth-scale.md      # アナグリフ描画の奥行き設計メモ
```

## 開発メモ

- 依存パッケージはありません。
- ゲームロジック、描画、スタイルは `index.html` にまとまっています。
- `localStorage` にベストスコアを保存します。
- アナグリフ描画の奥行き設計は `docs/depth-scale.md` を参照してください。
